Steps to Deploy Backend on Render
1. Prepare Your Backend Project
Ensure your backend (Node.js / Express / etc.) runs locally:
npm start
Make sure you have:
package.json
start script:
"scripts": {
  "start": "node index.js"
}
2. Push Code to GitHub
Initialize Git:
git init
git add .
git commit -m "initial commit"
Push to GitHub:
git remote add origin <repo_url>
git push -u origin main
3. Create Account on Render
Go to Render
Sign up / login using GitHub
4. Create New Web Service
Click New → Web Service
Connect your GitHub repository
Select your backend repo
5. Configure Settings

Fill these details:

Name → your project name
Environment → Node
Build Command:
npm install
Start Command:
npm start
6. Add Environment Variables (Important)
Go to Environment tab
Add variables like:
PORT
MONGO_URI
JWT_SECRET
7. Handle Port Properly

In your code:

const port = process.env.PORT || 5000;
app.listen(port);
8. Deploy
Click Create Web Service
Wait for build & deployment
9. Access Backend
You’ll get a URL like:
https://your-app.onrender.com
Auto Deployment
Every push to GitHub:
git add .
git commit -m "update"
git push

➡️ Render auto-deploys
